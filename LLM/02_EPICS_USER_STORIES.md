# MVP-Scope – Epics & User Stories

> **Quelle:** PRD v1.1, Abschnitt 3

---

## Epic 1: Audio-Aufnahme

| ID | User Story | Priorität |
|----|-----------|-----------|
| A1 | Als User möchte ich eine Audioaufnahme starten, damit ich meine Gedanken oder ein Gespräch festhalten kann. | Must |
| A2 | Als User möchte ich mein iPhone sperren und die Aufnahme im Hintergrund weiterlaufen lassen, damit ich mich auf die Situation konzentrieren kann. | Must |
| A3 | Als User möchte ich auf dem Sperrbildschirm sehen, dass eine Aufnahme läuft, und diese jederzeit stoppen können. | Must |
| A4 | Als User möchte ich über die Apple Watch eine Aufnahme starten, die zur Verarbeitung an mein iPhone gesendet wird. | Must |
| A5 | Als User möchte ich nach der Verarbeitung gefragt werden, ob die Originaldatei gelöscht werden soll, um Speicherplatz zu sparen. | Must |

## Epic 2: Transkription & Aufbereitung

| ID | User Story | Priorität |
|----|-----------|-----------|
| B1 | Als User möchte ich, dass meine Aufnahme nach dem Stoppen automatisch on-device transkribiert wird. | Must |
| B2 | Als User möchte ich automatisch eine Zusammenfassung meiner Aufnahme erhalten. | Must |
| B3 | Als User möchte ich automatisch extrahierte Action-Items sehen. | Must |
| B4 | Als User möchte ich eine automatisch generierte Kapitelübersicht für längere Aufnahmen erhalten. | Must |
| B5 | Als User möchte ich, dass automatisch Tags vergeben werden, die den Inhalt kategorisieren. | Must |

## Epic 3: Inhalte verwalten & bearbeiten

| ID | User Story | Priorität |
|----|-----------|-----------|
| C1 | Als User möchte ich eine listenförmige Übersicht aller meiner bisherigen Aufnahmen/Themen sehen. | Must |
| C2 | Als User möchte ich zwischen den Views Rohtranskript, Zusammenfassung, Action-Items und Kapitelübersicht wechseln können. | Must |
| C3 | Als User möchte ich Zusammenfassungen, Action-Items und Tags manuell bearbeiten können. | Must |
| C4 | Als User möchte ich über alle Inhalte hinweg suchen können, wobei Tags priorisiert werden. | Must |
| C5 | Als User möchte ich Inhalte als Markdown oder PDF exportieren können. | Must |

## Epic 4: Synchronisierung

| ID | User Story | Priorität |
|----|-----------|-----------|
| D1 | Als User möchte ich, dass meine Aufnahmen und deren Aufbereitungen über alle meine Apple-Geräte synchronisiert werden. | Must |
| D2 | Als User möchte ich mich dafür lediglich mit meinem Apple-Account anmelden – ohne separaten Account. | Must |

## Epic 5: Apple Watch

| ID | User Story | Priorität |
|----|-----------|-----------|
| E1 | Als User möchte ich auf meiner Apple Watch eine Aufnahme starten und stoppen können. | Must |
| E2 | Als User möchte ich, dass die Audio-Daten automatisch ans iPhone zur Verarbeitung übertragen werden. | Must |
| E3 | Als User möchte ich auf der Watch eine kompakte Übersicht meiner letzten Aufnahmen und deren Zusammenfassungen sehen. | Should |
