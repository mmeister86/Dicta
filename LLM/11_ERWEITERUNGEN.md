# Zukünftige Erweiterungsmöglichkeiten

> **Quelle:** PRD v1.1, Abschnitt 11

---

Die folgenden Features sind explizit **nicht** Teil des MVP, bieten aber Potenzial für zukünftige Versionen:

- **Benutzerdefinierte Prompts:** User können eigene Aufbereitungsanweisungen definieren (z.B. „Erstelle ein Protokoll im Format XY")
- **Sprecher-Erkennung (Diarization):** Automatische Zuordnung von Textpassagen zu verschiedenen Sprechern
- **Integration mit Apple Reminders / Kalender:** Action-Items direkt als Erinnerungen anlegen
- **Widgets (Home Screen / Lock Screen):** Schnellstart der Aufnahme, letzte Zusammenfassungen
- **Shortcuts-Integration:** Automatisierungen via Apples Shortcuts-App (z.B. nach jedem Meeting automatisch aufnehmen)
- **Weitere Exportformate:** DOCX, Integration mit Notiz-Apps (Apple Notes, Notion, Obsidian)
- **Erweiterte Suchfunktion:** Semantische Suche über alle Transkripte
- **Weitere Lokalisierungen:** Französisch, Spanisch, Italienisch etc. basierend auf Markt-Traction
- **visionOS-Support:** Potenzielle Erweiterung auf Apple Vision Pro
